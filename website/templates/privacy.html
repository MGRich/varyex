Privacy Policy Last updated June 4th, 2021

What data do you collect?

We collect:

  • profile data inputted by you via the profile command,
  • warning data when you warn a user and configure warns via the warn command,
  • the message ID, author ID, channel ID, and reaction count for a starboard message,
  • and the channel IDs for daily Garfield and SROMG strips.

Why do you need this data, and how do you use it?

Profile and warning data are stored for your convenience so you can actually see it. Starboard message data is to keep it as accurate as possible and gives Varyex the ability to easily fetch for the message if needed. The channel IDs for Garfield and SROMG are stored to ensure it goes into the right channel.

Who can see this data?

Only the developer (myself, Lulu) can directly fetch for the data.

How can we, as users, remove our data?

Profiles/parts of profiles and warns can both be deleted using commands, and you can unregister a channel from daily Garfield and SROMG strips. Starboard data gets removed on message deletion.

How do we reach to you if we have any concerns?

You can always join the support server if you have any questions or concerns and ask me about them directly.
